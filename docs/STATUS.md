## ✅ 1) المُنجز (مُطبّق ومنشور)

### الأمان والأداء
- إخفاء هوية المقاول على مستوى الخادم (`profiles_public` مُقنّع + `get_rfq_contractor`).
- `is_counterparty` داخل سياسة RLS → استبدلت بـ `EXISTS` مضمّن + فهارس (أداء).
- فترة تعتيم ضد القنص (آخر 30 دقيقة) في `get_rfq_offer_ranking`.
- دالة `cleanup_rate_limits` + عمود `approvals`.
- تحقّقات: عروض المنافسين/تسريب الأسعار/`handle_new_user` — سليمة.

### الذكاء و BOQ
- تقييد بنود BOQ بتصنيف من القوائم الرسمية.
- منع إزعاج الموردين (غير المصنّف لا يُرسل).
- درجة ثقة للتصنيف + مراجعة المنخفض.
- قراءة المواصفات ومطابقة الرموز (DRS-701) + مطابقة تقريبية.
- نموذج أرخص (Haiku) للتصنيف.

### Taxonomy في قاعدة البيانات (3 مراحل مكتملة)
1. جدول `taxonomy` + بذرة 117 تخصص + RLS + `get_taxonomy()`.
2. المصنّف الخادمي يقرأ كلمات الأدمن (كاش 5د) → توجيه بدون نشر.
3. الواجهات تقرأ الأسماء/التخصصات الجديدة (`hydrateTaxonomy` + `TaxonomyProvider`) + `/admin/taxonomy` كامل (تعديل اسم/كلمات + إضافة تخصص).

### الثقة والتصميم
- Supplier Score 0-100 يظهر للمقاول + شارات الاعتماد (أرامكو/نيوم...).
- ألوان القطاعات (الكهرباء كهرماني) + شريط تنقّل سفلي للجوال.
- تسمية الطلبات (نحو عربي سليم) + وسم «🟢 جديد» + حالات فارغة أوضح.

### خصوصية المورد والتواصل
- رقم واتساب المورد **لا يظهر للمقاول** — للمنصة/الإشعارات فقط.
- بعد الترسية فقط يظهر «رقم التواصل الرسمي» الذي يختاره المورد (أو واتساب إن لم يحدّد آخر).
- المورد يضبط رقم التواصل من الإعدادات: «نفس الواتساب» أو رقم مختلف.
- قبل الترسية: التواصل عبر الدردشة الداخلية فقط، بحدّ **٢٠ رسالة لكل محادثة** (منع الاتفاق خارج المنصة) — يُرفع الحدّ بعد الترسية.
- الدردشة الداخلية تُرسل فعلياً (عمود `body`/`read_at`) + زر رجوع + رسائل خطأ ظاهرة.
- سعر العرض في المصفوفة **شامل الضريبة** + زر «👁 التفاصيل» أوضح.

### ميزات وتجربة
- ترسية بند-بند (أرخص/أسرع/أقرب + تفاصيل المورد).
- تعديل/إلغاء العرض 15 دقيقة + تمديد المهلة الفوري.
- تواريخ على البطاقات + سجل أسعار + اتجاه السوق.
- نموذج تعديل الطلب (اسم/كمية/وحدة).
- تنقّل client-side (`next/link`) لكل التطبيق.
- **إزالة `@ts-nocheck` بالكامل (56 ملف) — strict نظيف، البناء أخضر.**
- توثيق: `PROJECT_DOSSIER.md` + صور + `REVIEW_RESPONSE.md`.

### الكتالوج والصفحات العامة (SEO)
- **ملف مراجعة الكتالوج** (`docs/CATALOG_REVIEW.md` + `catalog.json`) — 541 مادة مع تصنيفها ووحدتها وحالتها (مخصّص/عام) لعرضها على ذكاء آخر. يُولّد آلياً بـ`scripts/gen-catalog.cjs`.
- **مواصفات منظّمة** للأبواب/الشبابيك/الواجهات (16 مادة، صارت 61 مخصّصة) — نفس نمط الحديد.
- **`/suppliers`** لوحة شرف الموردين العامة (درجة موثوقية ٠–١٠٠ + ترتيب + شارات) — SSR/ISR.
- **`/prices`** مؤشّر أسعار مواد البناء العام (من صفقات الترسية) — SSR/ISR + نص SEO.
- `sitemap.ts` + `robots.ts` + ترويسة/تذييل عام (`PublicHeader`/`PublicFooter`).
